# VCL Icon Exporter

Figma plugin for batch exporting icon variants as PNG or SVG files into a ZIP archive.

## How It Works

### 1. Selection
Select one or more elements in Figma:
- **Component Set** — all variants inside will be exported
- **Component** — exports a single component
- **Instance** — exports the instance using the main component's naming

### 2. Export Options

| Option | Description |
|--------|-------------|
| **Format** | Choose between PNG or SVG export |
| **Include size 24 in file name** | When enabled, adds `24` suffix to file names for size-24 variants. Otherwise, size 24 is omitted from the name |
| **Create additional size 128 from size 32** | Automatically upscales size-32 variants to 128px (4× scale). Only available for PNG format |

### 3. File Naming

Exported files follow this pattern:

```
[PATH]/[BASENAME][SIZE]_[FORMAT].[ext]
```

- **PATH** — derived from the Section name containing the component. Sections with `/` in the name create nested folders
- **BASENAME** — sanitized component name (uppercase, no spaces or special characters)
- **SIZE** — numeric size from variant properties (extracted from the `size` property or any numeric property)
- **FORMAT** — `PNG` or `SVG`

**Example:**  
A component `Icon/Star` in section `Navigation/Actions` with size 24 will be exported as:
```
Navigation/Actions/ICONSTAR24_PNG.png
```

### 4. Archive

All files are packaged into a ZIP archive named:
```
YYYY_MM_DD_VCL_EXPORT_[FORMAT].zip
```

Click **Save ZIP** to download the archive.

## Technical Details

- The plugin uses a custom ZIP implementation without external dependencies
- Exports are prepared asynchronously with caching for better performance
- Size-128 variants are generated by exporting size-32 components at 4× scale

## Installation

1. Open Figma Desktop
2. Go to **Plugins** → **Development** → **Import plugin from manifest...**
3. Select the `manifest.json` file from this folder

## Usage

1. Select components or component sets to export
2. Run the plugin: **Plugins** → **VCL Icon Exporter**
3. Configure export options
4. Click **Save ZIP**
